### プロパティと値を追加する
コンストラクタの中で「this.プロパティ = 値」とすることで、生成されたインスタンスにプロパティと値を追加することができる。

### インスタンスとプロパティ
コンストラクタの中で追加した値はオブジェクトと同様に「インスタンス.プロパティ」とすることでクラスの外で使用することができる。
